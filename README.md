# Laravel Digging 👨‍💻

## About Repo 📖

This repo contains recaps for the Laravel book You will find it <a href="https://github.com/mahmoudmohamedramadan/Backend-Roadmap/blob/main/books/Matt%20Stauffer%20-%20Laravel_%20Up%20%26%20Running_%20A%20Framework%20for%20Building%20Modern%20PHP%20Apps-O%E2%80%99Reilly%20Media%20(2019).pdf">here</a>. You'll find these recaps in the file itself and every topic was explained in one file with a descriptive name such as you will find <a href="https://github.com/mahmoudmohamedramadan/Laravel-Digging-v8/blob/main/routes/new_features.php">new_features.php</a> file, This file contains the syntax in the new Laravel versions and so on. You'll find also some images from the book in the `public` folder.

> This project is compatible WITH less or equal version 8

## Installation 🎉

### Clone the repo

```git
git clone https://github.com/mahmoudmohamedramadan/Laravel-Digging-v8 Laravel_Digging_v8
```

### Go into the project's directory

```shell
cd Laravel_Digging_v8
```

### Install the project's dependencies

```composer
composer install
```
